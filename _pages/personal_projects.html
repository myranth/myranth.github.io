Cavey Jones

Java

Cavey Jones is my first attempt at coding a game from scratch. It is a coin-collecting platformer with tricky jumps and things designed to kill you, inspired by some older platformers along with more recent, difficult ones.

Lands of Terin

C++, OpenGL, SDL

Lands of Terin is a program that procedurally generates and renders 3D landscapes in real time. The worlds are made up of different biomes which form different land shapes and spawn different plants/trees. Other features include varying LOD (Level of Detail) terrain, reflective water, particles, and shadow mapping.

Cluster Chord

Python, Keras

Cluster Chord is the name of my fourth year group project, which was an investigation into the generation of music (in MIDI format) via the use of deep, recurrent neural networks. I was responsible for a lot of backend code, including transforming MIDI files into data suitable for a neural network (and back), and training networks with various layouts/parameters.
