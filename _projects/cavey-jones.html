About

Cavey Jones is a 2D platformer written in Java with Eclipse. It is heavily inspired by the Neopets game Hannah and the Pirate Caves, which I spent many hours creating levels for when I was younger. This was my first attempt at writing a game from scratch. All of the physics/game logic is built in, rather than using a game engine or library for assistance.

Each level involves collecting the coins and reaching the exit door, while avoiding various sharp things that kill you. There are 14 levels (plus an unfinished one). The game gets quite difficult quite quickly, and later levels may well take a lot of attempts.

Download

Download (zip file)

The zip file should contain a folder called res/ and a file called cavey.jar. You can run the program with:
java -jar cavey.jar

Github

You can view the code on github - it's pretty old and fairly messy! You can run it locally by cloning the repo:
git clone https://github.com/myranth/caveyJones.git
